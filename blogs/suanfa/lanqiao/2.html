    # 方格分割

    标签：DFS

    # 题目

    6x6的方格，沿着格子的边线剪开成两部分。 要求这两部分的形状完全相同。

    如下就是三种可行的分割法

    试计算： 包括这 33 种分法在内，一共有多少种不同的分割方法。 注意：旋转对称的属于同一种分割法。

    # 运行限制

    • 最大运行时间：1s
    • 最大运行内存: 128M

    # 思路

    以图像的中心点出发，沿着实线对称前进实现图像分割。

    建立坐标系：以左下角为原点，出发点为（3，3）

    # 代码

    #include <iostream>
using namespace std;
int ans;
//方向数组
int dire[][2]{{-1,0},
              {1,0},
              {0,-1},
              {0,1}};

//标记数组
int vis[7][7];

void dfs(int x,int y){
//当线走到边界时即为一个结果
  if(x==0||y==0||x==6||y==6){
    ans++;
    return;
  }
//当前的点标注为已访问
	vis[x][y]=1;

//对称的点也标注为已访问
	vis[6-x][6-y]=1;

//点往四个方向走
    for(int k=0;k<4;k++){
      int nx=x+dire[k][0];
      int ny=y+dire[k][1];
//新座标
      if(nx<0||nx>6||ny<0||ny>6) continue; //本题可以不用此句去判断越界
      if(!vis[nx][ny]) dfs(nx,ny);
    }
//回溯
    vis[x][y]=0;
    vis[6-x][6-y]=0;
}

int main()
{
  dfs(3,3);
  cout<<ans/4;
  return 0;
}
