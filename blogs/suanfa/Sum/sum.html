    # 一、判断闰年

    闰年： 对于非100倍数的年份，是4的倍数就是闰年。对于是100倍数的年份，需要是400的倍数才是闰年

    另一种记忆方式：

    ​ （4位数年份）若后两位是00则看前两位是不是4的倍数，例如1900不是闰年。否则看后两位数是不是4的倍数就行。

    模板：

    bool run(int y){
    return (y%100!=0&&y%4==0)||(y%400==0);
}

    # 四、二分搜索的题目特征

    二分题目一些比较明显的特征：在给定条件最大/小的前提下，求最大/小答案等（可以在答案可能的区间里直接搜索符合条件的答案）

    若答案区间具有单调性，则一定可以用二分查找；但如果不具有单调性，也有可能可以用二分做

    整数二分模版：

    bool check(int x) {/* ... */} // 检查x是否满足某种性质

// 区间[l, r]被划分成[l, mid]和[mid + 1, r]时使用：
int bsearch_1(int l, int r)
{
    while (l < r)
    {
        int mid = l + r >> 1;
        if (check(mid)) r = mid;    // check()判断mid是否满足性质
        else l = mid + 1;
    }
    return l;
}
// 区间[l, r]被划分成[l, mid - 1]和[mid, r]时使用：
int bsearch_2(int l, int r)
{
    while (l < r)
    {
        int mid = l + r + 1 >> 1;
        if (check(mid)) l = mid;	//注意当条件为l=mid时，mid需要先+1
        else r = mid - 1;
    }
    return l;
}

double bsearch_3(double l, double r)
{
    const double eps = 1e-6;   // eps 表示精度，取决于题目对精度的要求
    while (r - l > eps)
    {
        double mid = (l + r) / 2;
        if (check(mid)) r = mid;
        else l = mid;
    }
    return l;
}

    方法二思路：

    1. 递归第一层：让第一个数不同，得到n个数列（把第1个数和后面的每个数交换即可）

      1 2 3 4 ... n

      2 1 3 4 ... n

      3 2 1 4 ... n

      ...

      n 2 3 4 ... 1

      以上n个数列，只要第一个数不同，不管后面n-1个数是怎么排列的，这n个数列都不同

    2. 递归第二层：在上面的每个数列中，去掉第一个数，对后面的n-1个数进行类似的排列

      2 3 4 5 ... n

      3 2 4 5 ... n

      4 3 2 5 ... n

      ...

      n 3 4 5 ... 2

      以上n-1个数列，只要第一个数不同，不管后面n-2个数是怎么排列的，这n-1个数列都不同

    3. 继续以上过程，直到用完所有数字

    总共有：n*（n-1） *（n-2） * ... * 1个情况

    # 十三、一维差分

    差分： 指给定一个数组a，对a的每个元素a[i]都存在b[i] = a[i] - a[i - 1]。对任何 b1 一直加到 bi 都存在相加的结果等于ai，所以当 b 是 a 的差分数组时，a 也是 b 的前缀和数组。

    作用： 想要对 a 某一区间的值进行多次修改的话，可以用O（1）的复杂度修改 b 数组再求 b 数组的前缀和得到 a 数组

    **模板：**给数组a[]的区间[l, r]中的每个数加上c：

    b[l] += c, b[r + 1] -= c
    1

    差分的修改操作可以理解为当 b[l] += c 时，数组 a 从 l 开始的每个数都加了 c ，要想 l 到 r 区间 +c，还需要在 b[r + 1] -= c，从而使 r 后面的数不受影响。

    例如： 对于数组 a : 1 2 3 4 5 ，可以得到差分数组 b ：1 1 1 1 1； 如果想要对第3位到第4位进行加2操作，l = 3，r = 4，b[l] += 2，b[r + 1] -= 2; 这时的 b 数组：1 1 3 1 -1，进行前缀和，a = 1 2 5 6 5。正好对应之前操作。（进行乘除操作可以类推到前缀乘）

    # 十七、朴素筛法求素数

    素数： 也叫质数，在大于1的自然数中，除了1和它本身以外不再有其他因数

    朴素筛法模板：

    int primes[N], cnt;     // primes[]存储所有素数
bool st[N];         // st[x]存储x是否被筛掉，默认初始化为false（0）

//调用函数得到小于n的所有素数，存储在数组primes中
void get_primes(int n)
{
    for (int i = 2; i <= n; i ++ )
    {
        if (st[i]) continue;
        primes[cnt ++ ] = i;
        //将是i的倍数全都筛掉
        for (int j = i + i; j <= n; j += i)
            st[j] = true;
    }
}

    补充知识：“取模”运算的运算法则：

    1. (a + b) % p = (a % p + b % p) % p
    2. (a - b) % p = (a % p - b % p ) % p
    3. (a * b) % p = (a % p * b % p) % p

    借助第三个法则，只需要在循环乘积的每一步都提前进行“取模”运算，而不是等到最后直接对结果“取模”，也能达到同样的效果

    /**

 * @param power  指数
 * @return  求幂结果的最后3位数表示的整数
 */
long long normalPower(long long base,long long power){
    long long result=1;
    for(int i=1;i<=power;i++){
        result=result*base;
        result=result%1000;
    }
    return result%1000;

    long long result = 1;

    # 二十三、不能凑出的最大数

    类似题目：

    小明开了一家糖果店。

    他别出心裁：把水果糖包成4颗一包和7颗一包的两种。

    糖果不能拆包卖。

    小朋友来买糖的时候，他就用这两种包装来组合。

    当然有些糖果数目是无法组合出来的，比如要买 10 颗糖。

    你可以用计算机测试一下，在这种包装情况下，最大不能买到的数量是17。

    大于17的任何数字都可以用4和7组合出来。

    本题的要求就是在已知两个包装的数量时，求最大不能组合出的数字。

    如果 a,b 均是正整数且互质，那么由 ax+by,x≥0,y≥0 不能凑出的最大数是 ab−a−b

using namespace std;

typedef long long LL;

int main()

    LL a, b;

    cout << a * b - a - b << endl;

    return 0;

    # 2、输入转换：sscanf

    sscanf定义：

    int sscanf (const char *str, const char * format,........);

    函数说明：

    sscanf将str字符串以format格式进行转换，并将转换的结果存于对应的参数内。

    返回值：

    成功则返回参数数目；

    失败则返回-1；

    实例：

     int a, b, c;
 sscanf("2006:03:18", "%d:%d:%d", &a, &b, &c);	//对读入的字符串进行数据提取

    # 二、Tire树

    Trie树，又叫字典树、前缀树（Prefix Tree）、单词查找树 或 键树，是一种多叉树结构。

    上图是一棵Trie树，表示了关键字集合 (opens new window){“a”, “to”, “tea”, “ted”, “ten”, “i”, “in”, “inn”} 。从上图可以归纳出Trie树的基本性质：

    1. 根节点不包含字符，除根节点外的每一个子节点都包含一个字符。
    2. 从根节点到某一个节点，路径上经过的字符连接起来，为该节点对应的字符串。
    3. 每个节点的所有子节点包含的字符互不相同。

    通常在实现的时候，会在节点结构中设置一个标志，用来标记该结点处是否构成一个单词（关键字）。

    可以看出，Trie树的关键字一般都是字符串，而且Trie树把每个关键字保存在一条路径上，而不是一个结点中。另外，两个有公共前缀的关键字，在Trie树中前缀部分的路径相同，所以Trie树又叫做前缀树（Prefix Tree）

    模板：

    int son[N][26], cnt[N], idx;
// 0号点既是根节点，又是空节点
// son[][]存储树中每个节点的子节点
// cnt[]存储以每个节点结尾的单词数量

// 插入一个字符串
void insert(char *str)
{
    int p = 0;
    for (int i = 0; str[i]; i ++ )
    {
        int u = str[i] - 'a';
        if (!son[p][u]) son[p][u] = ++ idx;
        p = son[p][u];
    }
    cnt[p] ++ ;
}

// 查询字符串出现的次数
int query(char *str)
{
    int p = 0;
    for (int i = 0; str[i]; i ++ )
    {
        int u = str[i] - 'a';
        if (!son[p][u]) return 0;
        p = son[p][u];
    }
    return cnt[p];
}
