    数据类型注意事项

    默 12/11/2021 java

    # 一、java里数字的字面量默认为int类型

    # 各数据类型字节大小补充：1字节（byte）=8比特（bit）

    1. byte：1 【-128~127】（取值范围）
    2. short：2 【-32768~32767】
    3. int：4 【-2147483648~2147483647】
    4. long：8
    5. Boolean：1
    6. char：2（一个中文正好占2个字节） 【0~65535】
    7. float：4
    8. double：8

    若需要定义int i=2147483648，编译报错，整数过大，需写成int=2147483648L或int=2147483648l，建议用大写L，小写l长的像1

    # 二、类型转换

    1、小数据类型数据可自动转换为大数据类型数据，如

    byte i=127;
int j=i;		//byte可自动转int
    1
    2

    2、如果整数型字面量没有超出数据类型范围，那么这个字面量可以直接赋值给变量

    byte i=127;			//可以，int型字面量127会自动转换为byte类型
byte i=128;			//报错
char i=65535;		//可以，自动转换为char类型

    3、大数据类型数据转换为小数据类型数据需强制转换（可能会损失精度）

    int i=1;
byte j=(byte)i;			//即使“1”在byte范围内，但还是必须用这种方式强制转换，不然报错
						//虽然i的值没有超过byte类型范围，但编译器只会判断i是int类型，不知道i储存的值

    # 三、运算过程中的类型转换

    # 注意：

    # 多种数据类型做混合运算时，各自先转换成”最大容量“对应的类型（有long数据类型）

    # 但byte、char、short做混合运算时，会各自先转换成int再做运算

    char c1='a';
byte b=1;

//此处‘+’是负责求和
System.out.println(c1+b);	//输出结果为98

//报错：不兼容的类型：从int转换到short可能会有损失
short s=c1+b;			//虽然结果98没有超过short范围，但编译器只会判断c1+b是int类型，不知道计算结果

//正确写法：
short s=(short)(c1+b);

    # 四、任意一个浮点型都比整数型空间大

    eg：

    long类型占用8个字节

    float类型占用4个字节

    但是，float容量>long容量

    # 五、任何一个浮点型字面量数据默认当作double来处理

    eg：

    1.0 那么1.0默认为double类型

    1.0f或1.0F才是float类型

    float f=3.14;				//报错
float f=3.14f;
float f=3.14F;
float f=(float)3.14;

    补充：10.0/5的数据结果为double类型

    # 六、boolean类型数据只有两个值

    在c和c++语言中，0和1也可以表示布尔类型

    而在java中，boolean类型只有两个值：true和false

    # 注：八种基本数据类型，只有boolean类型不能进行类型转换
