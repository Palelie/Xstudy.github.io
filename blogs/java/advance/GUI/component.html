    有两个⽅法可以实现⼀个Icon

    ① 使⽤java.awt.Graphics类提供的功能来画图；

    ② 使⽤ Java⽀持的图⽚⽂件类型进⾏创建。

    1、使⽤java.awt.Graphic类提供的功能来画图

    class MyIcon implements Icon {
     private int width;
     private int height;
     public int getIconHeight() {
    	 return height;
	 }	
     public int getIconWidth() {
     	return width;
     }
     public MyIcon(int width, int height) {
         this.width = width;
         this.height = height;
     }
 	public void paintIcon(Component arg0, Graphics arg1, int x, int y) {
 		arg1.fillOval(x, y, width, height);
 	}
};

    # 七、菜单栏

    # 创建二级菜单栏：

    1、创建菜单条 JMenuBar

    2、创建菜单 JMenu

    3、创建菜单项 JMenuItem

    4、菜单添加菜单项

    5、菜单栏添加菜单

    总结: 菜单项依附菜单，菜单依附菜单条

    示例：

    JFrame jf = new JFrame("Frame");
 jf.setBounds(400,300,300,100);
//创建菜单条 JMenuBar
 JMenuBar bar=new JMenuBar();

//创建菜单 JMenu
 JMenu menu=new JMenu("菜单1");
 JMenu menu2=new JMenu("菜单2");

//创建菜单项 JMenuItem
 JMenuItem item1 = new JMenuItem("⼆级菜单1");
 JMenuItem item2 = new JMenuItem("⼆级菜单2");
 JMenuItem item3 = new JMenuItem("⼆级菜单3");
 JMenuItem item4 = new JMenuItem("⼆级菜单1");
 JMenuItem item5 = new JMenuItem("⼆级菜单2");
 JMenuItem item6 = new JMenuItem("⼆级菜单3");

//菜单添加菜单项
 menu.add(item1);
 menu.add(item2);
 menu.add(item3);
 menu2.add(item4);
 menu2.add(item5);
 menu2.add(item6);

//菜单栏添加菜单
 bar.add(menu);
 bar.add(menu2);

 jf.add(bar);
 jf.setVisible(true);
 jf.setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE);
