    # 常用面板

    ⾯板也是⼀种swing容器，他可以作为容器容器添加容纳其他的组件，但是他必须被加在⼀个容器内。

    # 一、JPanel

    JPanel就是⼀种最简单的⾯板，他继承⾃ java.awt.Container 类

    //创建窗口 
JFrame jf = new JFrame("Frame");
 jf.setBounds(400, 300, 1200, 1200);
 jf.setVisible(true);
//创建按钮
 JButton jb = new JButton("⾹蕉");
 JButton jb2 = new JButton("草莓");
// ⽣成⼀个⾯板
 JPanel jp = new JPanel(new FlowLayout()); 	//流式布局
//把按钮放到面板上
 jp.add(jb);
 jp.add(jb2);
// 将⾯板加⼊到另外的容器（窗口）中
 jf.add(jp);

    常见绘图方法：（示例）

    import javax.swing.*;
import java.awt.*;

/**
 * 如何在画板上画出图形
 */


public class DrawCircle extends JFrame{ //JFrame对应窗口，可以理解成一个画框

    //定义一个面板
    private MyPanel01 mp = null;

    public static void main(String[] args) {
        new DrawCircle();
    }
    
     //构造方法
    public DrawCircle(){   
        //初始化面板
        mp = new MyPanel01();
        //把面板放入到窗口（画框）
        this.add(mp);
        //设置窗口大小
        this.setSize(400,300);
        //可以显示
        this.setVisible(true);
        //设置关闭
        this.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
    }
}

//定义一个MyPanel，继承JPanel类，相当一个画板，画图就在画板上画
class MyPanel01 extends JPanel{
//    说明：
//    1、MyPanel 对象就是一个画板
//    2、Graphics g 把 g 理解成一支画笔
//    3、Graphics 提供了很多绘图方法

//    paint 该方法在创建对象时会自动调用
    @Override
    public void paint(Graphics g) { //绘图方法
        super.paint(g); //调用父类的方法完成初始化

        //常用绘制不同图形的方法
        //直线
        g.drawLine(10,10,100,100);
        
        //矩形边框
        g.drawRect(10,10,100,100);
        
        //画一个圆形（椭圆）
        g.drawOval(10,10,100,100);
        
        //填充矩形
        g.setColor(Color.blue); //设置画笔颜色
        g.fillRect(10,10,100,100);
        
        //填充椭圆
        g.setColor(Color.blue); //设置画笔颜色
        g.fillOval(10,10,100,200);
        
        //插入图片
        //1、获取图片资源   (图片在项目根目录下)
        Image image = Toolkit.getDefaultToolkit().getImage(Panel.class.getResource("/pc.png"));
        g.drawImage(image,10,10,175,221,this);
        
        //画字符串(写字)
        g.setColor(Color.blue); //设置画笔颜色
        g.setFont(new Font("宋体",Font.BOLD,50));    //设置字体
        g.drawString("你好",100,100); //起始坐标在左下角

    }
}

    # 二、JScrollPane

    JScrollPane是带滚动条的⾯板。因此，在碰到需要将⼀个⼤的组件放到⼀个较⼩的空间的时候，使⽤该⾯板。JScrollPane内只能添加⼀个组件；因此，如果需要将多个组件添加到⼀个JScrollPane中时，需要先将需要添加的组件添加到JPanel中，再将JPanel添加到JScrollPane中。

     JFrame jf = new JFrame("Frame");
 jf.setBounds(400, 300, 1200, 200);
 jf.setVisible(true);
 JButton jb = new JButton("⾹蕉");
 jb.setSize(800,200);
 JScrollPane jp=new JScrollPane(jb);
 jf.add(jp);
